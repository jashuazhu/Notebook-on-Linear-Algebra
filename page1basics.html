Vector Foundations

Core vector operations with matching worked examples.

1) Vectors & Components

A vector in $\mathbb{R}^n$ is $\mathbf{x} = (x_1, \dots, x_n)^\top = \sum_{i=1}^n x_i \mathbf{e}_i$, where $\mathbf{e}_i$ are the standard basis vectors.
Example in $ \mathbb{R}^3 $ : $ \mathbf{x} = (2, -1, 4)^\top = 2\mathbf{e}_1 - \mathbf{e}_2 + 4\mathbf{e}_3 $.

2) Addition & Scalar Multiplication

Vector addition and scalar multiplication:

\[ \mathbf{u} + \mathbf{v} = (u_1 + v_1, \ldots, u_n + v_n)^\top, \qquad c\mathbf{v} = (c v_1, \ldots, c v_n)^\top. \]

Properties: commutativity, associativity, and distributivity:

\[ \mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}, \quad (\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w}), \quad c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}. \]
Let $\mathbf{u} = (1, -2, 3)$, $\mathbf{v} = (4, 0, -1)$, $c = 2$.
$\mathbf{u} + \mathbf{v} = (1+4, -2+0, 3-1) = (5, -2, 2)$.
$c\mathbf{u} = (2, -4, 6)$.

3) Linear Combinations & Span

\[ \alpha\mathbf{u} + \beta\mathbf{v}, \quad \mathrm{span}\{\mathbf{u}, \mathbf{v}\} = \{ \alpha\mathbf{u} + \beta\mathbf{v} \mid \alpha,\beta\in\mathbb{R} \}. \]
With $\mathbf{u} = (1, -2, 3)$, $\mathbf{v} = (4, 0, -1)$, $(\alpha, \beta) = (2, -1)$:
$2\mathbf{u} - \mathbf{v} = (2, -4, 6) - (4, 0, -1) = (-2, -4, 7)$.

4) Dot Product & Angle

\[ \mathbf{u}\cdot\mathbf{v} = \sum_{i=1}^n u_i v_i = \mathbf{u}^\top\mathbf{v}, \qquad \cos\theta = \frac{\mathbf{u}\cdot\mathbf{v}}{\|\mathbf{u}\|\|\mathbf{v}\|}. \]
$\mathbf{u} = (1, -2, 3)$, $\mathbf{v} = (4, 0, -1)$.
$\mathbf{u}\cdot\mathbf{v} = 1\cdot4 + (-2)\cdot0 + 3\cdot(-1) = 1$.
$\|\mathbf{u}\| = \sqrt{14}$, $\|\mathbf{v}\| = \sqrt{17}$, so $\cos\theta = \frac{1}{\sqrt{14}\sqrt{17}}$.

5) Norms & Distance

\[ \|\mathbf{x}\|_2 = \sqrt{\sum_i x_i^2}, \quad \|\mathbf{x}\|_1 = \sum_i |x_i|, \quad \|\mathbf{x}\|_\infty = \max_i |x_i|. \] Distance: $d(\mathbf{x}, \mathbf{y}) = \|\mathbf{x} - \mathbf{y}\|$.
For $\mathbf{u} = (1, -2, 3)$:
$\|\mathbf{u}\|_2 = \sqrt{14}$, $\|\mathbf{u}\|_1 = 6$, $\|\mathbf{u}\|_\infty = 3$.

6) Unit Vectors & Orthonormality

Normalize: $\hat{\mathbf{x}} = \frac{\mathbf{x}}{\|\mathbf{x}\|}$. Orthonormal set $\{\mathbf{q}_i\}$ satisfies $\mathbf{q}_i^\top\mathbf{q}_j = \delta_{ij}$.
Normalize $\mathbf{v} = (4, 0, -1)$: $\hat{\mathbf{v}} = \frac{1}{\sqrt{17}}(4, 0, -1)$.

7) Projection onto a Vector

\[ \mathrm{proj}_{\mathbf{v}}(\mathbf{u}) = \frac{\mathbf{u}\cdot\mathbf{v}}{\mathbf{v}\cdot\mathbf{v}}\,\mathbf{v}. \]
$\mathbf{u} = (1, -2, 3)$, $\mathbf{v} = (4, 0, -1)$:
$\mathbf{u}\cdot\mathbf{v} = 1$, $\mathbf{v}\cdot\mathbf{v} = 17$.
$\mathrm{proj}_{\mathbf{v}}(\mathbf{u}) = \frac{1}{17}(4, 0, -1) = \left(\tfrac{4}{17}, 0, -\tfrac{1}{17}\right)$.

8) Cross Product (in $\mathbb{R}^3$)

\[ \mathbf{u}\times\mathbf{v} = (u_2v_3 - u_3v_2,\; u_3v_1 - u_1v_3,\; u_1v_2 - u_2v_1), \] and $\|\mathbf{u}\times\mathbf{v}\| = \|\mathbf{u}\|\|\mathbf{v}\|\sin\theta$.
$\mathbf{u} = (1, -2, 3)$, $\mathbf{v} = (4, 0, -1)$. $\mathbf{u}\times\mathbf{v} = (2, 13, 8)$.

9) Triangle Inequality & Parallelogram Law

\[ \|\mathbf{u} + \mathbf{v}\| \le \|\mathbf{u}\| + \|\mathbf{v}\|, \qquad \|\mathbf{u}+\mathbf{v}\|^2 + \|\mathbf{u}-\mathbf{v}\|^2 = 2\|\mathbf{u}\|^2 + 2\|\mathbf{v}\|^2. \]
$\mathbf{u} = (1, -2, 3)$, $\mathbf{v} = (4, 0, -1)$.
$\|\mathbf{u}+\mathbf{v}\| = \sqrt{33} \le \sqrt{14} + \sqrt{17}$ (true).
$33 + 29 = 62 = 2(14) + 2(17)$.

10) Componentwise (Hadamard) Operations

Componentwise product: $(\mathbf{u}\circ\mathbf{v})_i = u_i v_i$.
$\mathbf{u} = (1, -2, 3)$, $\mathbf{v} = (4, 0, -1)$.
$\mathbf{u}\circ\mathbf{v} = (4, 0, -3)$.

Summary Properties

  • Linearity: $(\alpha\mathbf{u}+\beta\mathbf{v})\cdot\mathbf{w}=\alpha(\mathbf{u}\cdot\mathbf{w})+\beta(\mathbf{v}\cdot\mathbf{w})$.
  • Orthogonality: $\mathbf{u}\perp\mathbf{v}\iff \mathbf{u}\cdot\mathbf{v}=0$.
  • Normalization: $\hat{\mathbf{u}}=\mathbf{u}/\|\mathbf{u}\|$.
  • Projection: $\mathrm{proj}_{\mathbf{v}}(\mathbf{u})=(\mathbf{u}\cdot\mathbf{v})/(\mathbf{v}\cdot\mathbf{v})\mathbf{v}$.
  • Cross product is perpendicular to both operands in $\mathbb{R}^3$.
