Chapter 2

Matrix Fundamentals

Matrix arithmetic, determinants, inverses, and elimination routines.

1) Matrix Addition & Scalar Multiplication

For $A,B\in\mathbb{R}^{m\times n}$ and scalar $c$: \[ (A+B)_{ij} = A_{ij}+B_{ij}, \qquad (cA)_{ij} = c\,A_{ij}. \] Properties: \[ A+B = B+A,\quad (A+B)+C = A+(B+C),\quad c(A+B)=cA+cB. \]
Let \[ A=\begin{bmatrix}1&2\\3&4\end{bmatrix},\quad B=\begin{bmatrix}-1&0\\5&2\end{bmatrix},\quad c=2. \] Then \[ A+B=\begin{bmatrix}0&2\\8&6\end{bmatrix},\qquad cA=\begin{bmatrix}2&4\\6&8\end{bmatrix}. \]

2) Matrix Multiplication

If $A \in\mathbb{R}^{m\times n}$ and $B \in\mathbb{R}^{n\times p}$, then \[ (AB)_{ij} = \sum_{k=1}^{n} A_{ik} B_{kj}. \] Not commutative in general: $AB \\ne BA$.
With \[ A=\begin{bmatrix}1&2\\3&4\end{bmatrix},\quad B=\begin{bmatrix}-1&0\\5&2\end{bmatrix}, \] \[ AB=\begin{bmatrix} 1(-1)+2(5) & 1(0)+2(2)\\ 3(-1)+4(5) & 3(0)+4(2) \end{bmatrix} = \begin{bmatrix}9&4\\17&8\end{bmatrix}. \] But \[ BA= \begin{bmatrix} -1&0\\5&2 \end{bmatrix} \begin{bmatrix} 1&2\\3&4 \end{bmatrix} = \begin{bmatrix} -1&-2\\11&18 \end{bmatrix} \ne AB. \]

3) Transpose

\[ (A^\top)_{ij} = A_{ji},\qquad (AB)^\top = B^\top A^\top,\qquad (A^\top)^\top = A. \]
If \[ A=\begin{bmatrix}1&2\\3&4\end{bmatrix},\quad B=\begin{bmatrix}-1&0\\5&2\end{bmatrix}, \] then \[ A^\top=\begin{bmatrix}1&3\\2&4\end{bmatrix},\quad (AB)^\top=\begin{bmatrix}9&17\\4&8\end{bmatrix}^\top=\begin{bmatrix}9&4\\17&8\end{bmatrix} =B^\top A^\top. \]

4) Trace

\[ \operatorname{tr}(A)=\sum_{i=1}^{n} A_{ii},\qquad \operatorname{tr}(AB)=\operatorname{tr}(BA). \]
For \( A=\begin{bmatrix}1&2\\3&4\end{bmatrix} \), \( \operatorname{tr}(A)=1+4=5. \) Using the $AB$ above, \[ \operatorname{tr}(AB)=9+8=17,\quad \operatorname{tr}(BA)=-1+18=17. \]

5) Determinant & Laplace Expansion

For $2\times2$: \[ \det\!\begin{bmatrix}a&b\\c&d\end{bmatrix}=ad-bc. \] Laplace (cofactor) expansion along row $i$: \[ \det(A)=\sum_{j=1}^{n} (-1)^{i+j} A_{ij}\, \det(M_{ij}), \] where $M_{ij}$ is the minor after removing row $i$ and column $j$. Properties: \[ \det(AB)=\det(A)\det(B),\quad \det(A^\top)=\det(A). \]
(i) $2\\times2$: For \( A=\begin{bmatrix}1&2\\3&4\end{bmatrix} \), \( \det(A)=1\cdot4-2\cdot3=-2. \)

(ii) $3\\times3$ via Laplace along the first row: \[ C=\begin{bmatrix} 2&1&0\\ -1&3&4\\ 0&5&1 \end{bmatrix},\quad \det(C)= 2\det\!\begin{bmatrix}3&4\\5&1\end{bmatrix} -1\det\!\begin{bmatrix}-1&4\\0&1\end{bmatrix} +0\cdot(\cdots). \] Compute minors: \( \det\!\begin{bmatrix}3&4\\5&1\end{bmatrix}=3\cdot1-4\cdot5=-17 \), \( \det\!\begin{bmatrix}-1&4\\0&1\end{bmatrix}=-1\cdot1-4\cdot0=-1. \) Hence \( \det(C)=2(-17)-1(-1)=-34+1=-33. \)

6) Inverse

$A^{-1}$ satisfies $AA^{-1}=A^{-1}A=I$ and exists iff $\det(A)\neq0$. For $2\\times2$: \[ A^{-1}=\frac{1}{\det(A)}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}. \]
For \( A=\begin{bmatrix}1&2\\3&4\end{bmatrix} \), \( \det(A)=-2 \), \[ A^{-1}=\frac{1}{-2}\begin{bmatrix}4&-2\\-3&1\end{bmatrix} =\begin{bmatrix}-2&1\\1.5&-0.5\end{bmatrix}. \] Check: \[ AA^{-1}=\begin{bmatrix}1&2\\3&4\end{bmatrix}\!\begin{bmatrix}-2&1\\1.5&-0.5\end{bmatrix} =\begin{bmatrix}1&0\\0&1\end{bmatrix}=I. \]

7) Rank

$\operatorname{rank}(A)$ is the dimension of the column space (equals row space dimension). \[ \operatorname{rank}(A) = \text{number of pivots after row-reduction}. \]
Let \[ R=\begin{bmatrix} 1&2&3\\ 2&4&6\\ 1&1&1 \end{bmatrix}. \] Note row 2 $=2\\times$ row 1, so it is dependent. After elimination, two pivot rows remain $\Rightarrow \operatorname{rank}(R)=2$.

8) Elementary Matrices (for Gaussian Elimination)

Three types (each corresponds to a single row operation on $I$):
  • Row swap $R_i\leftrightarrow R_j$
  • Row scaling $R_i\leftarrow c\,R_i$ ($c\neq0$)
  • Row add $R_i\leftarrow R_i + c\,R_j$
Applying an elementary matrix $E$ to $A$ on the left performs that row operation: $EA$.
Start with \( A=\begin{bmatrix}1&2\\3&4\end{bmatrix}. \)

Row swap ($R_1\\leftrightarrow R_2$): \[ E_{\text{swap}}=\begin{bmatrix}0&1\\1&0\end{bmatrix},\quad E_{\text{swap}}A=\begin{bmatrix}3&4\\1&2\end{bmatrix}. \] Row scaling ($R_1\\leftarrow 2R_1$): \[ E_{\text{scale}}=\begin{bmatrix}2&0\\0&1\end{bmatrix},\quad E_{\text{scale}}A=\begin{bmatrix}2&4\\3&4\end{bmatrix}. \] Row add ($R_2\\leftarrow R_2 - 3R_1$): \[ E_{\text{add}}=\begin{bmatrix}1&0\\-3&1\end{bmatrix},\quad E_{\text{add}}A=\begin{bmatrix}1&2\\0&-2\end{bmatrix}. \] Chaining these yields $PA=LU$ in Gaussian elimination (with permutation $P$).

Summary Properties

  • $(AB)^\top = B^\top A^\top$, $\operatorname{tr}(AB)=\operatorname{tr}(BA)$.
  • $\det(AB)=\det(A)\det(B)$, $\det(A^\top)=\det(A)$, $A$ invertible $\iff \det(A)\neq 0$.
  • $\operatorname{rank}(A)$ = number of pivots (invariant under row operations).
  • Elementary matrices implement single row operations: left-multiplication $EA$.
